# Mai

## Prerequisites

Install the following dependencies (if not done yet):

1. git
2. python

## Setup

1. Clone this repository

2. Change directory with `cd mai`

3. Run the following commands to set up.

   - MacOS: `sh setup.sh`
   - Windows: `.\setup.bat`

## Usage

1. Run the following command.

   - MacOS: `sh run.sh`
   - Windows: `.\run.bat`
